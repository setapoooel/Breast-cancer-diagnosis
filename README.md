# 🎗️ Breast Cancer Diagnosis Using Machine Learning

A machine learning project developed in Python for breast cancer diagnosis using clinical tumor measurement data.

This project utilizes the Breast Cancer Wisconsin Dataset and applies Logistic Regression to classify tumors as benign or malignant. The workflow includes data preprocessing, feature normalization, model training, and performance evaluation.

---

## 📌 Features

* Breast cancer diagnosis using machine learning
* Logistic Regression classification model
* Data preprocessing and cleaning
* Feature normalization
* Training and testing dataset split
* Model accuracy evaluation
* Simple and easy-to-understand implementation

## 🧠 Machine Learning Workflow

1. Load the breast cancer dataset.
2. Remove unnecessary columns.
3. Convert diagnosis labels:

   * Malignant (M) → 1
   * Benign (B) → 0
4. Normalize input features.
5. Split data into training and testing sets.
6. Train a Logistic Regression model.
7. Evaluate prediction performance on test data.
